⚡ Aurora Solo Scrypt Miner

One-page LTC + DOGE solo-mining control build. It follows the same Control Nest idea as the BTC page: RPC template pull → worker hashing → candidate detection → submitblock audit.

One HTML page Litecoin Core RPC Dogecoin Core RPC Web Workers Solo / regtest-first

Testing note: Use regtest/testnet first. This build uses the local Aurora Scrypt Gateway and a pure-JavaScript scrypt_1024_1_1_256 worker path. Mainnet solo odds are lottery-level and browser Scrypt is slow compared with ASICs.

🌉 Local CORS Gateway

GitHub Pages cannot directly fetch Litecoin/Dogecoin Core RPC. This build now talks to a local gateway instead.

LTC endpoint: http://127.0.0.1:8788/ltc
DOGE endpoint: http://127.0.0.1:8788/doge

Run the gateway with: node aurora-scrypt-gateway.js
